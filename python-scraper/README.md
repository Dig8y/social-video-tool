## Installation

1. **Clone the repository:**

   ```bash
   git clone <repository-url>
   cd <repository-directory>
   ```

2. **Prepare Virtual Environment:**

   ```bash
    python -m venv venv
    source venv/bin/activate
   ```

3. **Install Dependences:**

   ```bash
      pip install -r requirements.txt
   ```
